## Recent Releases

Review the latest versions below, ordered newest first. Each release includes new features, improvements, and fixes tailored to enhance your documentation management experience.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>
## New Features

- Added real-time collaboration for shared documentation spaces
- Introduced custom branding options including brand color support (e.g., `#007BFF`)
- New export formats: PDF and Markdown bundles

## Improvements

- Enhanced search with fuzzy matching and full-text indexing
- Optimized page load times by `>30%`
- Better mobile responsiveness for documentation previews

## Bug Fixes

- Fixed navigation issues in nested hierarchies
- Resolved image upload failures in certain browsers
</Update>

## Upgrade Instructions

Follow these steps to update your Sourcingmentor instance.

<Steps>
  <Step title="Backup" icon="database">
    Create a full backup of your documentation space.
    
    ```bash
    sourcingmentor backup --all
    ```
  </Step>
  <Step title="Update" icon="download">
    Run the update command with the latest version.
    
    <CodeGroup tabs="CLI,Docker">
    ````bash
    sourcingmentor update v1.2.0
    ````
    ````bash
    docker pull sourcingmentor:latest
    docker-compose up -d
    ````
    </CodeGroup>
  </Step>
  <Step title="Verify" icon="check-circle">
    Test key features like search and collaboration.
    
    Visit `https://dashboard.example.com` to confirm.
  </Step>
</Steps>
