---
title: Cannot Upload File
description: Fix issues when uploading files to Sourcingmentor.
---

## Symptoms

- File upload fails or stalls
- Error: "Upload failed"

## Solutions

<Steps>
  <Step title="Check file size">
    Maximum file size is 50MB. Reduce your file size if needed.
  </Step>
  <Step title="Verify file type">
    Only PDF, DOCX, and XLSX formats are supported.
  </Step>
  <Step title="Try another browser">
    Upload in Chrome or Firefox if problem persists.
  </Step>
</Steps>
